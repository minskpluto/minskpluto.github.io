Weekly review

Parabola formula: y = ax^2 + bx + c

To find the y intercept, make x = 0 to find the value of y.

To find the x intercept, make y = 0 to find the value of x

To find the vertex, find the midpoint of two x intercepts, and substitute them into y = ax^2 + bx + c (parabola formula) Then, find the y value. Therefore, that y value and the x value is the vertex.

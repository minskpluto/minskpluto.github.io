Simplifying fraction using expansion

In a fraction, when the denominator is a algebraic equation, you must expand the algebraic equation to make 2 fractions have the same denominator.

For example, if a denominator of 2 fractions are v^2 -4 and v+2, you must expand v^2 -4, which will be (v+2)(v-2)

You can see that both denominators have (v+2), which is the lowest common denominator. Therefore, the lowest common denominator of these 2 fractions is (v+2)(v-2)

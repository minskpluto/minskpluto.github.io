Geometrically, a function is an even function if its graph is symmertrical (or reflected) about the y-axis

The algerbraical definition of an even function y=f(x) is: y=f(-x) = f(x)

Geometrically, a function is an odd fucntion if its graph is symmertrical about the x axis. The algebraical definiton of an odd function y=f(x) is: y+ f(-x) + -f(x) for all real x.

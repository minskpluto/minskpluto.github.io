What is logarithm?

In mathematics, the logarithm is the inverse function to exponentiation. That means the logarithm of a given number x is the exponent to which another fixed number, the base b, must be raised, to produce that number x.

Exponents and Logarithms

An [exponent] tells us how many times the base need t multiply itself to get another number. For example, 2*4 = 16 tells us that 2 needs to be multiplied [four] times to get 16. Mind you, the power is 4.

logaX + logaY = loga[XY]

Proof - we know that a*X + a*Y = a*X+Y. Now, let m = a*X and n = a*Y. Hence, mn = a*X+Y. Change it into logarithmic form : loga(mn) = X+Y, and so logaX + logaY = logaXY
